# School Timetable Solver – Project Brief

Objective: Build a timetable generator for classes 6–12 using Python.
AI (Claude) is used only for reasoning about constraints and heuristics,
not for manipulating full timetable grids.

Key principles:
- Separate data, constraints, solver logic
- Avoid full timetable grids during reasoning
- Represent schedule elements as events
- Use dry runs + validation reports
- Generate Excel only at the end
